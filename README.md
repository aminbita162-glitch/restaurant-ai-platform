# Restaurant AI Plattform

Dies ist eine intelligente Restaurant-Plattform mit KI-Unterstützung.

## Funktionen
	- Digitale Speisekarte
	- KI-basierte Empfehlungen
	- Online-Bestellung
	- Verwaltungssystem für Restaurants

## Ziel

Ziel dieses Projekts ist die Entwicklung einer modernen, KI-gestützten Lösung für Restaurants.
